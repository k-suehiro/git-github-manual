現在行っている修正で、まだコミットをしていない内容を確認します。

$ git status
On branch add-new-file
Changes to be committed:
  (use "git reset HEAD ..." to unstage)
    
    new file:   sample.txt

先ほど add したので、新規ファイルとして sample.txt が認識されています。

もし、add を忘れていると、以下のようにコミットの対象にならない Untracked files として認識され、修正内容をコミットすることができません。

$ git status
On branch add-new-file
Untracked files:
  (use "git add ..." to include in what will be committed)
    
    sample.txt
